小程序的缓存

18-06-20 / 23 Words
  1. 判断本地存储中有没有旧的数据。{time:Date.now(),data:[. ]}
  2. 没有旧的数据则重新发送请求
  3. 有旧的数据同时旧的数据没有过期则使用本地存储
const Cates=wx.getStorageSync(“cates”)
If(!Cates){
this.getCates()
}else{
            If(Date.now()-Cates.time)>1000*10){
                                this.getCates()
                }else{
                            this.Cates=Cates.data
                            let leftMenuList=this.Cates.map(v=>v.cat_name)
                            let rightContent=this.Cates[0].children
                            this.setData({
                                leftMenuList
                                rightContent
                            })
                        }
}
